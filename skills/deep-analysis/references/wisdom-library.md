# 大师著作库

> **使用原则**(v2.0.0 更新):
> - 大师是**可选武器**,不是必填格式。无匹配不引用。
> - 引用必须服务分析(能多穿透一层才引),不为装饰贴名言。
> - 允许直接用思想不贴名言;允许两位对撞呈现分歧。
> - 引用时注明出处著作,配一句"为什么这句话在这里起作用"。
> - **不受限于本库**:AI 应根据素材自由召唤任何合适的思想家(老子、费曼、达尔文、马基雅维利、德鲁克、凯恩斯、福柯、彼得·圣吉……按需,不列举)。本库只是原五位大师的著作速查,便于在这五人里找匹配时用。
